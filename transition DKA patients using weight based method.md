# transition DKA patients using weight based method

- related: [[DKA diabetic ketoacidosis and HHS]]
- tags: #permanent
---

Transition DKA patients using 0.2 - 0.4 u/kg as weight based method:

- Use 0.2 u/kg for insulin naive patients
	- 0.1 u/kg for long and short acting each
- Use 0.4 u/kg for insulin resistant diabetes patients
	- 0.2 u/kg for long and short acting each
