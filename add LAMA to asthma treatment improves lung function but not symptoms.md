# add LAMA to asthma treatment improves lung function but not symptoms

- related: [[Asthma reactive airway disease]]
- tags: #permanent
---

Adding LAMA to ICS-LABA regimen for asthma treatment increases lung function (~80mL) and decreased exacerbations but does not improve symptoms.[^1]

[^1]: https://ginasthma.org/pocket-guide-for-asthma-management-and-prevention/
