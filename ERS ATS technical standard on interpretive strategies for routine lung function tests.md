# ERS ATS technical standard on interpretive strategies for routine lung function tests

- related: [[PFT and lung functions]]
- tags: #literature
---

- GLI reference used over NHANES from previous recommendation in 2005
- GLI has more age included
- lung volumes must be considered for pts with BMI > 40: can have both obstructive and restrictive disease
- use LLN and z-score over percent predicted and FEV/FVC ratio
- use 10% increase relative for predicted FEV1 or FVC for bronchodilator response

[^1]

[^1]: Stanojevic S. ERS/ATS technical standard on interpretive strategies for routine lung function tests. _EUROPEAN RESPIRATORY JOURNAL_.
